# 🐍 Snake Game

A simple **Snake Game** built using **Python** with the `pygame` library.

## 🎮 Features
- Classic snake gameplay
- Increasing difficulty as the snake grows
- Simple and interactive UI
- Score tracking

## 🛠 Technologies Used
- **Python**
- **Pygame** (for game graphics and mechanics)
